## Overview

The schedules system (`working-horarios`) manages different time slots for desk reservations. The system supports three default schedules: morning, afternoon, and full day.

## Endpoint

```
GET /api/working-horarios
```

Base URL: `https://macfer.crepesywaffles.com`

## Schedule Types

The system defines three schedule IDs:

<ParamField query="id" type="number">
  Schedule identifier:
  - `1` - **Mañana** (Morning: 8:00 AM - 12:00 PM)
  - `2` - **Tarde** (Afternoon: 1:00 PM - 5:00 PM)
  - `3` - **Día completo** (Full day: 8:00 AM - 5:00 PM)
</ParamField>

## Verification Window

Each schedule has a 25-minute verification window from the start time.

<ParamField query="verificationWindowMinutes" type="number" default="25">
  Time window (in minutes) after shift start when users can confirm their reservation
</ParamField>

### Verification Constants

```javascript
const VERIFICATION_WINDOW_MINUTES = 25;

// Example: Morning shift (8:00 AM)
// Verification window: 8:00 AM - 8:25 AM
// After 8:25 AM, pending reservations are auto-cancelled
```

## Extract Schedule from Reservation

Get the schedule ID from a reservation object:

```javascript
const extractId = (rel) => {
  if (rel === null || rel === undefined) return null;
  if (typeof rel === 'number') return rel;
  if (typeof rel === 'object' && rel.id !== undefined) return rel.id;
  
  if (rel.data !== null && rel.data !== undefined) {
    const d = rel.data;
    if (typeof d === 'number') return d;
    if (Array.isArray(d) && d.length > 0) return d[0].id ?? null;
    if (typeof d === 'object' && d.id !== undefined) return d.id;
  }
  
  return null;
};

const getHorarioId = (reserva) => {
  return (
    extractId(reserva.attributes?.working_horarios) ??
    extractId(reserva.working_horarios) ??
    null
  );
};
```

## Schedule Conflicts

When a desk has multiple reservations, certain schedule combinations block others:

```javascript
const turnosBloqueados = (reservas, puestoId) => {
  const bloq = new Set();
  
  reservas
    .filter(r => getPuestoId(r) === puestoId && esReservaActiva(r))
    .forEach(r => {
      const hId = getHorarioId(r);
      
      if (hId === 1) {        // Morning blocks full day
        bloq.add(1);
        bloq.add(3);
      } else if (hId === 2) { // Afternoon blocks full day
        bloq.add(2);
        bloq.add(3);
      } else if (hId === 3) { // Full day blocks everything
        bloq.add(1);
        bloq.add(2);
        bloq.add(3);
      }
    });
  
  return bloq;
};
```

**Conflict Rules:**
- Morning reservation blocks: Morning + Full Day
- Afternoon reservation blocks: Afternoon + Full Day
- Full Day reservation blocks: All schedules

## Auto-Cancellation

Pending reservations are automatically cancelled if not confirmed within the verification window:

```javascript
const AUTO_CANCEL_REASON = '! Usuario no confirmo!';

// Check if verification window has passed
const isPast = timeInfo.isPast;

if (isPast && reserva.estado === 'Pendiente') {
  // Auto-cancel the reservation
  await updateReservaWithVerification(reserva.id, {
    estado: 'Cancelada',
    confirmada: false,
    motivoCancelacion: AUTO_CANCEL_REASON,
    verificacionAsistencia: {
      fecha: new Date().toISOString(),
      mensaje: AUTO_CANCEL_REASON,
      tipo: 'auto-cancelacion',
    },
  });
}
```
