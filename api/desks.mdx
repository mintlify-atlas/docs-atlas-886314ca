## Overview

The desk management system handles individual desk positions (`working-puestos`) and rooms (`working-salas`). Each desk belongs to a room and can be assigned specific features like monitors.

## Desk Positions (Puestos)

## Get Desk Status

Check desk availability for a specific date.

### Calculate Desk State

```javascript
// Desk states: 'disponible', 'limitado', 'ocupado'
const calcEstado = (reservas, puestoId) => {
  const rp = reservas.filter(r => 
    getPuestoId(r) === puestoId && 
    esReservaActiva(r)
  );
  
  const tieneAM = rp.some(r => getHorarioId(r) === 1);
  const tienePM = rp.some(r => getHorarioId(r) === 2);
  const tieneCompleto = rp.some(r => getHorarioId(r) === 3);
  
  if (tieneCompleto || (tieneAM && tienePM)) return 'ocupado';
  if (tieneAM || tienePM) return 'limitado';
  return 'disponible';
};
```

**States:**
- `disponible` - Desk is completely free
- `limitado` - Desk is reserved for one shift (morning or afternoon)
- `ocupado` - Desk is fully booked

### Get Available Desks

```javascript
import { getEscritoriosDisponibles } from './utils/reservasService';

// Get available desk IDs for a date
const availableDesks = await getEscritoriosDisponibles('2026-03-15');
// Returns: [1, 2, 4, 5] (array of available desk IDs)
```

**Implementation:**

```javascript
export const getEscritoriosDisponibles = async (fecha) => {
  const reservas = await getReservas({ fecha });
  const activas = reservas.filter((reserva) => 
    reserva.estado === 'Pendiente' || 
    reserva.estado === 'Confirmada'
  );
  
  const ocupados = activas
    .map((reserva) => Number(reserva.escritorioId))
    .filter((id) => !Number.isNaN(id));
  
  return [1, 2, 3, 4, 5, 6].filter((id) => !ocupados.includes(id));
};
```

## Rooms (Salas)

Rooms are workspaces that contain multiple desks.

### Endpoint

```
GET /api/working-salas
```

Base URL: `https://macfer.crepesywaffles.com`

### Request

```http
GET /api/working-salas?populate=foto&sort=id:asc
```

### Query Parameters

<ParamField query="populate" type="string">
  Populate the `foto` field to get room images
</ParamField>

<ParamField query="sort" type="string">
  Sort results (e.g., `id:asc`)
</ParamField>

### Response

<ResponseField name="data" type="array">
  Array of room objects
  
  <ResponseField name="id" type="number">
    Unique room identifier
  </ResponseField>
  
  <ResponseField name="attributes" type="object">

    <ResponseField name="capacidad" type="number">
      Total number of desk positions in the room

    <ResponseField name="monitores" type="number">

### Normalize Room Data

```javascript
const normalizeSala = (item) => {
  const a = item.attributes ?? item;
  
  // Extract image URL
  let imagenUrl = null;
  const fotoArr = a.foto?.data;
  if (Array.isArray(fotoArr) && fotoArr.length > 0) {
    const url = fotoArr[0].attributes?.url;
    if (url) {
      imagenUrl = url.startsWith('http') 
        ? url 
        : `https://macfer.crepesywaffles.com${url}`;
    }
  }
  
  return {
    id: item.id,
    nombre: a.nombre ?? a.name ?? `Sala ${item.id}`,
    capacidad: a.capacidad ?? 0,
    monitores: a.monitores ?? 0,
    imagen: imagenUrl,
    disponible: a.disponible !== false,
    ruta: '/reservas',
  };
};
```

## Working with Desk Relations

Reservations link to desks through the `working_puestos` relation.

### Extract Desk ID from Reservation

```javascript
const extractId = (rel) => {
  if (rel === null || rel === undefined) return null;
  if (typeof rel === 'number') return rel;
  if (typeof rel === 'object' && rel.id !== undefined) return rel.id;
  
  if (rel.data !== null && rel.data !== undefined) {
    const d = rel.data;
    if (typeof d === 'number') return d;
    if (Array.isArray(d) && d.length > 0) return d[0].id ?? null;
    if (typeof d === 'object' && d.id !== undefined) return d.id;
  }
  
  return null;
};

const getPuestoId = (reserva) => {
  return (
    extractId(reserva.attributes?.working_puestos) ??
    extractId(reserva.working_puestos) ??
    null
  );
};
```

### Get Room Information from Reservation

```javascript
const getSalaInfo = (reserva) => {
  const puestosData = reserva.attributes?.working_puestos?.data;
  const pAttr = Array.isArray(puestosData)
    ? puestosData[0]?.attributes
    : puestosData?.attributes ?? null;
    
  const salaData = pAttr?.working_sala?.data;
  
  if (salaData) {
    return {
      salaId: salaData.id ?? null,
      salaNombre: salaData.attributes?.nombre ?? `Sala ${salaData.id}`,
    };
  }
  
  return { salaId: null, salaNombre: null };
};
```

## Blocked Shifts

Check which shifts are blocked for a desk:

```javascript
const turnosBloqueados = (reservas, puestoId) => {
  const bloq = new Set();
  
  reservas
    .filter(r => getPuestoId(r) === puestoId && esReservaActiva(r))
    .forEach(r => {
      const hId = getHorarioId(r);
      if (hId === 1) { // Morning
        bloq.add(1);
        bloq.add(3); // Full day also blocked
      } else if (hId === 2) { // Afternoon
        bloq.add(2);
        bloq.add(3);
      } else if (hId === 3) { // Full day
        bloq.add(1);
        bloq.add(2);
        bloq.add(3);
      }
    });
  
  return bloq;
};
```
